Actualmente, pertenezco al programa Jóvenes Construyendo el Futuro, donde estoy recibiendo capacitación en el área Front End, donde uso html, css y javascript para la parte de desarrollo web, para la creación y rediseño de páginas web de clientes de la empresa.
